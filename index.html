Nuestros Servicios

Concreto diseñado para cada método de colocación

Tiro Directo

Entrega directa desde el camión revolvedor. Ideal para losas, cimientos y proyectos con acceso vehicular.

Tiro Bombeado

Bombas de 28m y 36m de alcance. Solución para edificios, zonas de difícil acceso y proyectos en altura.

Fabricamos concreto con especificaciones para aplicación con equipos de lanzado neumático (shotcrete). Ideal para taludes, túneles y superficies irregulares.
