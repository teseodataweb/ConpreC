## 3. ALCANCE TÉCNICO DETALLADO DE LOS SERVICIOS

### 3.1 SERVICIO 1: ConcreteOS - Cotizador Avanzado

#### 3.1.1 Descripción Funcional

Sistema web de cotización interna que permite a los vendedores de Conpre Concretos generar cotizaciones precisas, personalizadas y profesionales en tiempo real, considerando todas las variables del producto (resistencia, tipo de tiro, tiempos de fraguado, extras) y del cliente (ubicación, tipo de cliente, condiciones comerciales).

#### 3.1.2 Requisitos de Negocio

**Confidencialidad Crítica:**
- El cotizador **NO debe ser público**
- Acceso exclusivo para personal autorizado de Conpre
- Los precios son información sensible y estratégica

**Usuarios del Sistema:**
- Vendedores de Conpre Concretos
- Personal administrativo autorizado
- Gerencia (con métricas y reportes)

#### 3.1.3 Campos y Variables Requeridas

**Datos del Cliente:**
- RFC
- Nombre/Razón Social
- Tipo de Cliente (Autoconstrucción / Desarrollos / Obra Pública / Otro)
- Zona de Contacto (para cálculo de transporte)
- Método de Pago (contado, crédito, anticipos)
- Datos de Contacto (teléfono, email, dirección)

**Configuración del Producto:**
- Categoría de Producto:
  - Concreto Premezclado (F'c)
  - Concreto para Pavimentos (MR)
  - Morteros
  - Rellenos Fluidos
  - Productos Personalizados
- Resistencia/Especificación (según categoría)
- Tipo de Tiro (Directo / Bombeado / Lanzado)
- Tiempo de Fraguado (Normal 28 días / Rápida 14, 7, 3, 2, 1 día)
- Extras/Modificadores (impermeabilizante, fibra, grava, color, revenimiento, etc.)
- Volumen (m³)

**Salida del Cotizador:**
- Cotización en formato PDF profesional con branding de Conpre
- Desglose de costos (unitario, subtotal, IVA, total)
- Condiciones comerciales
- Vigencia de la cotización
- Datos de contacto de Conpre

#### 3.1.4 Requisitos Técnicos

**Tipo de Aplicación:** WebApp responsiva (multi-dispositivo)
**Compatibilidad:** Desktop, tablet, smartphone
**Hosting:** Por definir con cliente (recomendación: HostGator México, Plan Emprendedor ~$551.25 MXN/año)
**Autenticación:** Sistema de login con roles y permisos
**Base de Datos:** Almacenamiento de cotizaciones, clientes, productos, precios

**Métricas del Vendedor (Requerimiento Explícito):**
- Número de cotizaciones generadas por vendedor
- Tasa de conversión (cotizaciones → ventas)
- Volumen total cotizado vs. vendido
- Productos más cotizados

#### 3.1.5 Integraciones Futuras (Potencial)

- CRM para seguimiento de leads
- Sistema de remarketing vía WhatsApp
- Análisis econométrico de mercado
- Agente de prospección vertical con IA

### 3.2 SERVICIO 2: TCO (Total Cost of Ownership) por Proyecto

#### 3.2.1 Descripción Funcional

Calculadora de análisis Go/No-Go que permite a Conpre determinar si un proyecto específico es rentable antes de comprometerse comercialmente, considerando todos los costos asociados (producción, transporte, mano de obra, oportunidad, riesgos).

#### 3.2.2 Variables de Entrada (Estimadas)

- Volumen del proyecto (m³)
- Tipo de concreto y especificaciones
- Distancia al sitio de obra
- Condiciones de pago del cliente
- Tiempo de entrega requerido
- Costos de producción (materia prima, energía, mano de obra)
- Costos de transporte (combustible, depreciación de flota)
- Costos indirectos (administrativos, financieros)

#### 3.2.3 Salida Esperada

- **Margen de Rentabilidad del Proyecto:** Porcentaje de ganancia neta
- **Punto de Equilibrio:** Volumen mínimo para no perder
- **Recomendación Go/No-Go:** Decisión basada en criterios predefinidos
- **Análisis de Sensibilidad:** Impacto de variables críticas (distancia, precio de venta, volumen)

#### 3.2.4 Alcance del Plan Light

Según el documento comercial, el Plan Light incluye **TCO por Proyecto** (Go/No-Go), no el TCO completo por compañía (que está en Plan Premium). Esto implica que el enfoque es **análisis proyecto a proyecto**, no análisis integral de la operación empresarial.
